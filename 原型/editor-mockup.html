vsinkore Editor

A WYSIWYG Markdown editor that lives inside VSCode. The surface borrows the editor's own typography and colors, so it reads as part of the IDE rather than a separate app.

Inline formatting

Text supports bold, italic, inline code, underline, strikethrough, colored text, and links. Selecting a run reveals the bubble toolbar.

Block quotes use the editor's quote tokens for the bar and tinted background, matching the host theme in both light and dark.

Code block

typescript 
// Shiki highlight follows the active VSCode theme
export function activate(context: ExtensionContext) {
  const provider = new InkoreEditorProvider(context);
  context.subscriptions.push(
    window.registerCustomEditorProvider("inkore.editor", provider),
  );
  return 0;
}
